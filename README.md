# Testimonials Section

A simple and responsive Testimonials Section built using **HTML** and **CSS**.

## Features

* Responsive card layout
* Clean and modern design
* Customer testimonial cards
* Hover effects
* Easy to customize
* Beginner-friendly project

## Technologies Used

* HTML5
* CSS3

## Project Structure

```text
project-folder/
│
├── index.html
├── style.css
└── README.md
```

## Preview

This project displays customer testimonials for a website called **GearVize**. Each testimonial includes:

* Customer Name
* Location
* Star Rating
* Review Text

## Learning Objectives

Through this project, you will learn:

* HTML structure and semantic elements
* CSS Grid Layout
* Responsive Design
* Card Components
* Hover Animations
* Spacing and Typography

## Getting Started

1. Clone the repository

```bash
git clone <repository-link>
```

2. Open the project folder

3. Open `index.html` in your browser
